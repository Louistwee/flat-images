Flower / Grass Position Calculator

Guide

  1. Upload the image or use the example image to test it
  2. Select a block with the corners visible (don't use farmland)
  3. Place the Points in the corners of the block by changing the x/y values in the table
  4. Selec a flower (it doesn't have to be on the block you selected and don't use farmland)
  5. Bring the cursor point at the intersection of the center of the flower and the ground
  6. Read the position of the flower acording to the block where it is standing on below (use this information in your program)
  7. Repeat from step 4 (and see the example below)
